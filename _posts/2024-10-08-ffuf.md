# Multiple WordLists

```bash
ffuf -w <file.txt>:W1,<file2.txt>:W2 -X POST -d "username=W1&password=W2" -H "Content-Type: application/x-www-form-urlencoded" -u <url>
```
W1,W2 could be any word or letter.

# Using Pipes

```bash
seq 1 1000 | ffuf -w - -u http://URL/FUZZ
```
